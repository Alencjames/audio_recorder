# AudioRecorderApp

A simple iOS audio recorder app built with pure UIKit.

## Features

- **Record & Playback**: Record audio with live waveform visualizers.
- **Transcripts**: Live speech-to-text transcription of your voice recordings.
- **Calendar**: Filter your recordings by date using an inline calendar.
- **Settings**: Dynamic side menu with light and dark mode toggles.
- **Pure Code**: 100% programmatic Swift layout with no storyboards.

## Setup & Running

1. Double-click `AudioRecorderApp.xcodeproj` to open it in Xcode.
2. Select a simulator (e.g. iPhone 16) or a connected iOS device.
3. Press `Command + R` (or click the play button) to build and run.
